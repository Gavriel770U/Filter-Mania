# Filter-Mania
Set of various interesting and funny filters made in C and C++ using ofGraphics.

Filters available:
*  RGB negate (RGB -> 255-RGB)
*  Grayscale
*  Rainbow flag
*  Grayscale + Red detection + bold
*  Top half double
*  Top half mirror
*  Triangle mirror
*  Pixelation
*  Dark frame
*  White bottom glow
*  Stickman world (White-Black contrast)
